## 🌹 ¿Qué es Nino Nakano Bot?

**Nino Nakano** es un bot de WhatsApp modular y de alto rendimiento, diseñado bajo la arquitectura de **𝓐𝓪𝓻𝓸𝓶**. No es solo un bot común; está enfocado en la gestión de grupos, entretenimiento y personalización extrema con una estética cyberpunk/anime.

> "No creas que estoy aquí para ayudarte porque quiero... ¡solo asegúrate de usarme bien, tonto!" — *Nino.*

---

## ✨ Características Principales

* **Sistema de Plugins Dinámico:** Agrega o quita funciones sin romper el código base.
* **Gestión de Grupos Pro:** Comandos de administración como `#kick` y `#ban` con validación de seguridad.
* **Bienvenida Personalizada:** Saludos automáticos con la foto del usuario y menciones.
* **Sistema de Niveles & XP:** Ranking automático de usuarios basado en su actividad.
* **Vinculación Versátil:** Soporte para código de 8 dígitos o código QR.
* **Optimizado para Termux:** Bajo consumo de recursos y alta estabilidad.

## 🛠️ Instalación en Termux

Si quieres desplegar tu propia instancia, sigue estos pasos:

1. **Actualizar paquetes:**
   ```bash
   pkg update && pkg upgrade -y

 * Instalar dependencias de sistema:
   pkg install nodejs-lts git ffmpeg imagemagick -y

 * Clonar el repositorio:
   git clone [https://github.com/zoredevteam-ctrl/Nino-Nakano)
cd Nino-Nakano

 * Instalar módulos de Node:
   npm install

 * Iniciar el bot:
   npm start

⚙️ Configuración (settings.js)
Edita el archivo settings.js para personalizar tu experiencia:
 * global.ownerNumber: Tu número de administrador.
 * global.botName: El nombre que le darás a tu bot.
 * global.rcanal: El enlace a tu canal de WhatsApp.
 * global.banner: La imagen principal que aparecerá en los menús.
